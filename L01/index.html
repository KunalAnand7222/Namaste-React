//difference between ./style.css and style.css is that ./style.css is a relative path which means it will look for the file in the same directory as the index.html file, while style.css is an absolute path which means it will look for the file in the root directory of the project. // ./ vs ../ is used to navigate through the directory structure. ./ means the current directory, while ../ means the parent directory. So if you want to access a file that is in the same directory as the index.html file, you can use ./style.css. If you want to access a file that is in the parent directory, you can use ../style.css.
